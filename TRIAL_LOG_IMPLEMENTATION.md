# 試煉日誌功能實作完成報告

## 功能概述
成功實作「試煉日誌」功能，可記錄並顯示最近 10 次終極試煉的挑戰紀錄。

## 實作內容

### 1. UI 介面設計 ✅

#### 入口按鈕（JourneyMode）
- **位置**：「我的冒險旅程」標題下方、第一張卡片上方
- **樣式**：紅色漸層背景 + 紅色邊框 + 尖刺圖標（鑽石形狀）
- **文字**：「試煉日誌」（紅色像素字體）
- **動作**：點擊後執行 `setView('trial-log')`

#### 日誌頁面（TrialLogView）
- **外觀**：深紫色漸層背景 + 金色像素邊框
- **頂部區域**：
  - 左上角：返回按鈕（返回冒險旅程）
  - 中間：「試煉日誌」標題 + 歷史最高等級勳章
  - 最高等級勳章：圓形設計，顯示 S/A/B/C/D 等級，使用對應顏色
- **清單內容**：
  - 顯示最近 10 筆紀錄
  - 每筆紀錄包含：
    - 編號標籤（#1, #2...）- 黃色圓形徽章
    - 等級勳章（右上角）- 旋轉 12 度的圓形勳章
    - 日期（2026/01/20 格式）- 使用時鐘圖標
    - 單元組合（Ch 1, 2, 5 格式）- 使用書本圖標
    - 分數 - 使用獎盃圖標
  - hover 效果：卡片放大至 105%
- **底部**：「再次挑戰」按鈕
  - 紅橙漸層背景 + 金色邊框
  - 兩側劍圖標
  - 點擊直接跳轉至挑戰設定頁

#### 儲存機制
- **時機**：每次終極試煉（challenge-quiz）完成時
- **資料結構**：
```javascript
{
    timestamp: "2026-01-20T14:00:00.000Z",
    units: ["1", "2", "5"],  // 挑戰的單元ID陣列
```

- **儲存位置**：Firestore `users/{uid}/trialHistory`
- **限制**：只保留最新 10 筆記錄（使用 `.slice(0, 10)`）
- **順序**：新記錄添加到陣列頂部（使用 `[newRecord, ...currentHistory]`）

## 使用說明

### 查看試煉日誌
1. 登入遊戲
2. 點擊世界地圖右上角的「背包」圖標進入「我的冒險旅程」
3. 點擊頁面上方的紅色「試煉日誌」按鈕
4. 查看你的挑戰歷史記錄

### 進行終極試煉
1. 從世界地圖點擊左上角的「紅龍」圖標
2. 選擇要挑戰的單元（可多選）
3. 點擊「開始挑戰」
4. 完成挑戰後，記錄會自動保存

### 重新挑戰
1. 在試煉日誌頁面
2. 點擊底部的「再次挑戰」按鈕
3. 直接進入挑戰設定頁面
